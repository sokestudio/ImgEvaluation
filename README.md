# ImgEvaluation
全参考图像质量评价

## 相关库安装
- pip install numpy
- pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple
- pip install scikit-image==0.15.0 -U -i https://pypi.tuna.tsinghua.edu.cn/simple
